# EcoTrack 🌱

EcoTrack is a waste management tracking system that helps residents and administrators manage garbage collection efficiently.

## 🚀 Features
- User Signup & Login
- Attendance Tracking (Collection Status)
- Live Vehicle Tracking
- Report Issues
- Schedule Pickup
- Payment Integration (Razorpay)
- Notifications System
- Admin Dashboard

## 🛠 Tech Stack
- Frontend: React.js, CSS
- Backend: Node.js, Express.js
- Database: MySQL
- Payment: Razorpay

## 📂 Project Structure
- /src → Frontend
- /backend → Backend APIs
